# Common Python Function Decorators
A package holding commonly used decorators for Python functions. The first 5 decorators came from or were inspired by 
[5 Python Decorators I Use in Almost All My Data Science Projects ](https://medium.com/towards-data-science/python-decorators-for-data-science-6913f717669a) by [Thuwarakesh Murallie](https://thuwarakesh.medium.com/).
